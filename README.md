# TTX_sim
## 使用方法
<pre>
このリポジトリをクローンしてください。
「API_key.txt」を作りOpenAIからAPIキーを取得してAPI_key.txtに保存してください。
使用するライブラリはFlask、Flask-socketio、OpenAI、SQLite3
データベーステーブルは
chat
id, comment, time

incident
id, json(JSON-LD形式)
</pre>
